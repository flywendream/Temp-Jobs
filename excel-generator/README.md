# 服务报价单 Excel 生成器

根据「服务报价单」模板生成带公式的 Excel 文件。

## 使用方法

1. 安装依赖（任选一种方式）：
```bash
# 方式一：全局安装
pip install -r requirements.txt

# 方式二：安装到项目目录（无需系统权限）
pip install -r requirements.txt --target ./lib
```

2. 运行生成脚本：
```bash
python generate_quotation.py
```

3. 在当前目录会生成 `服务报价单.xlsx` 文件。

## 包含的公式

- **含税优惠合计**：`数量 × 含税优惠单价`（每行明细自动计算）
- **合计**：自动汇总所有明细行的含税优惠合计

## 表格结构

- 左侧：客户信息（客户编号、公司名称、地址、联系人等）
- 右侧：报价信息（报价单号、日期、税率、类型、付款方式、交货期等）
- 明细表：服务费用、备件费用、运输费用三个分类，可填写数量和单价后自动计算合计
